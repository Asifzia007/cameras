
# Camera Management Dashboard

A feature-rich, responsive web application built with React for managing cameras. This dashboard enables users to search, filter, update statuses, and manage camera data with a user-friendly interface.

## Features

* **Responsive Design** : Adapts seamlessly to different screen sizes, ensuring usability across devices.
* **Search Functionality** : Easily locate cameras by their name or location.
* **Filtering Options** : Narrow down results based on camera status (Active/Inactive) or location.
* **Pagination** : Navigate through a large dataset with an intuitive pagination system.
* **Status Update** : Toggle the status of cameras between Active and Inactive with real-time updates.
* **Delete Camera** : Remove unwanted cameras from the list after confirmation.

## Technologies Used

* **React** : Framework for building the interactive user interface.
* **CSS3** : Custom styles with responsive design using media queries.
* **JavaScript (ES6+)** : Core language for app functionality.
* **Fetch API** : Handle API requests for data retrieval and updates.
* **FontAwesome** : Icon library for visual enhancements (e.g., search, delete, sync).

## Prerequisites

Before you begin, ensure you have the following installed on your system:

* **Node.js** : [Download here](https://nodejs.org/) (LTS version recommended)
* **npm** : Comes bundled with Node.js
* **A Code Editor** : [Visual Studio Code](https://code.visualstudio.com/) is recommended.

## API Endpoints

### Fetch Cameras

* **Endpoint** : `https://api-app-staging.wobot.ai/app/v1/fetch/cameras`
* **Method** : `GET`
* **Headers** :
* `Authorization: Bearer <your-token>`

### Update Camera Status

* **Endpoint** : `https://api-app-staging.wobot.ai/app/v1/update/camera/status`
* **Method** : `PUT`
* **Headers** :
* `Content-Type: application/json`
* `Authorization: Bearer <your-token>`
